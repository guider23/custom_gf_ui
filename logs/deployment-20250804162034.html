How to Use This Form Generator

This tool allows you to create and deploy a custom form that submits data to a Google Form.

  • Step 1: Paste your Google Form Link
    • Copy the link to your Google Form (e.g., https://docs.google.com/forms/d/e/.../viewform).
    • Paste it into the "Google Form Link" field.
  • Step 2: Copy the JSON Extraction Code
    • Click the "Copy Code" button below to copy the JSON extraction code.
    • (function() { setTimeout(() => { let dataMapping = []; try { if (window.FB_PUBLIC_LOAD_DATA_ && Array.isArray(FB_PUBLIC_LOAD_DATA_[1][1])) { dataMapping = FB_PUBLIC_LOAD_DATA_[1][1].map(item => ({ id: item[4][0][0], question: item[1] })); } } catch (e) { console.log('Error parsing FB_PUBLIC_LOAD_DATA_:', e.message); } const results = []; document.querySelectorAll('input, select, textarea').forEach(element => { const entryName = element.getAttribute('name') || ''; if (entryName.startsWith('entry.')) { const id = entryName.split('.')[1]; const mapping = dataMapping.find(m => m.id === parseInt(id)); if (mapping) { results.push({ id, question: mapping.question }); } } }); console.log(JSON.stringify(results, null, 2)); }, 3000); })();
    • This code will run in the browser console to generate the JSON data needed for the form.
  • Step 3: Extract JSON Data
    • Go to your Google Form link, press Ctrl+Shift+I to open the developer tools, and navigate to the Console tab.
    • If prompted, type allow pasting and press Enter.
    • Paste the copied code into the console and press Enter.
    • After a few seconds, the console will output a JSON array of your form fields. Copy this JSON and paste it into the "JSON Data" field.
  • Step 4: Generate Form
    • Click the "Generate Form" button to create the form based on your JSON.
    • The form will appear with input fields matching your JSON data.
  • Step 5: Deploy and Share
    • If the form fields look correct, click the "Deploy" button to host your form on GitHub Pages.
    • Once deployed, view the deployment URL in the history panel (click the button).

Note

Google Form Settings > Response > Turn off Limit to One Response. Ensure your Google Form is set to accept responses and the entry IDs in your JSON match the form's fields.
